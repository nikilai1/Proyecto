# Ecomerce

Aplicacion nativa creada como proyecto final de la carrera de ***Desarrollo de aplicaciones moviles*** de ***CoderHouse***.

## Tecnologias Utilizadas
 * React Native
 * React
 * react native-web
 * react native-screens
 * react redux
